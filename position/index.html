CSS Position Cheatsheet

position: static; (default)

Static

Элементы с position: static; отображаются в обычном потоке документа. Свойства top, right, bottom, left и z-index не применяются.

position: relative;

Relative

Элемент с position: relative; позиционируется относительно своего нормального положения. Можно использовать top, right, bottom, left для смещения.

position: absolute;

Absolute

Элемент с position: absolute; позиционируется относительно ближайшего позиционированного предка (не static). Если такого предка нет, то относительно <body>.

position: fixed;

Fixed

Элемент с position: fixed; позиционируется относительно окна браузера. Он не прокручивается вместе со страницей.

z-index

z-index: 1
z-index: 2
z-index: 3
z-index: auto (0)

Свойство z-index управляет стековым порядком позиционированных элементов. Элемент с большим z-index будет выше элемента с меньшим.

position: sticky;

Sticky

Элемент с position: sticky; ведет себя как relative, пока не достигнет определенной точки при прокрутке, после чего "залипает" и ведет себя как fixed.
