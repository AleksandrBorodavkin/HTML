Шпаргалка по формам HTML

Тег <form>

Основной контейнер для всех элементов формы.

<form action="/submit-url" method="post" target="_blank" enctype="multipart/form-data" novalidate autocomplete="on">
  ... элементы формы ...
</form>
  • action: URL-адрес, на который отправляются данные формы.
  • method: HTTP-метод для отправки данных (get или post).
  • target: Определяет, где отображать ответ после отправки формы (_self, _blank, _parent, _top).
  • enctype: Тип кодирования данных при отправке (application/x-www-form-urlencoded, multipart/form-data, text/plain). Используйте multipart/form-data для загрузки файлов.
  • novalidate: Отключает встроенную проверку (валидацию) полей формы браузером.
  • autocomplete: Включает или отключает автозаполнение для всей формы (on или off).

Элементы ввода <input>

Самый универсальный элемент формы. Его поведение зависит от атрибута type.

Основные типы:

type="text"

Однострочное текстовое поле.

type="password"

Поле для ввода пароля (символы скрыты).

type="submit"

Кнопка для отправки формы.

type="button"

Просто кнопка (действие задается через JS).

type="reset"

Кнопка для сброса всех полей формы к их исходным значениям.

type="checkbox"

Флажок (чекбокс) для выбора нескольких опций.


type="radio"

Переключатель (радиокнопка) для выбора одной опции из группы (группируются по атрибуту name).


type="file"

Поле для выбора файла для загрузки на сервер.

Типы для ввода данных (HTML5):

type="date"

Поле для выбора даты.

type="datetime-local"

Выбор даты и времени (без часового пояса).

type="time"

Поле для выбора времени.

type="month"

Поле для выбора месяца и года.

type="week"

Поле для выбора недели и года.

type="email"

Поле для ввода email (с базовой валидацией).

type="number"

Поле для ввода числа.

type="tel"

Поле для ввода номера телефона (без специальной валидации).

type="url"

Поле для ввода URL-адреса.

type="search"

Текстовое поле для поисковых запросов.

type="range"

Ползунок для выбора числа в диапазоне.

type="color"

Элемент для выбора цвета.

Другие элементы форм

<textarea>

Многострочное текстовое поле.

<select>, <option>, <optgroup>

Выпадающий список.

<button>

Более гибкая кнопка, чем <input type="button">. Может содержать HTML.

<label>

Связывает текстовую метку с элементом формы. Улучшает доступность.

<fieldset> и <legend>

Группирует связанные элементы в форме.

Личные данные:

<datalist>

Создает список предопределенных опций для автозаполнения поля <input>.

<progress>

Индикатор выполнения задачи.

70%

<meter>

Измеритель значения в известном диапазоне.

60%

Важные атрибуты элементов

  • name: Имя элемента, используется для отправки данных на сервер.
  • value: Значение элемента.
  • placeholder: Подсказка в текстовых полях, исчезающая при вводе.
  • required: Указывает, что поле обязательно для заполнения.
  • disabled: Блокирует доступ к элементу. Данные не отправляются.
  • readonly: Поле нельзя редактировать, но его данные отправляются на сервер.
  • checked: Заранее выбирает `checkbox` или `radio`.
  • selected: Заранее выбирает `option` в `select`.
  • multiple: Позволяет выбрать несколько значений (`select`) или файлов (`input type="file"`).
  • min, max, step: Для числовых полей - минимальное, максимальное значение и шаг.
  • minlength, maxlength: Минимальная и максимальная длина для текстовых полей.
  • pattern: Регулярное выражение для проверки значения поля.
  • for: (у `label`) связывает метку с элементом по его `id`.
  • form: (у элемента) связывает элемент с формой по ее `id` (полезно, если элемент находится вне тега `
    `).
